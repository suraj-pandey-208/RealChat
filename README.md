 Real-Time Chat Application

A  full-stack real-time chat application built using the MERN Stack. The application enables secure one-to-one communication through JWT-based authentication and Socket.IO while providing a responsive and intuitive user experience.

Overview

This project demonstrates the implementation of a modern chat platform with secure authentication, real-time messaging, user presence tracking, media sharing, and efficient state management. It was developed to gain hands-on experience with scalable full-stack application development and real-time systems.

Key Features

Secure user authentication using JWT with protected routes and persistent sessions. Instant one-to-one messaging powered by Socket.IO. Real-time online/offline user presence with automatic status updates. Emoji support for interactive conversations. Image sharing through Cloudinary integration. User search for quick conversation access. Profile management with profile picture and username updates. Responsive user interface optimized for desktop and mobile devices.

Tech Stack

Frontend: React.js, Redux Toolkit, Tailwind CSS

Backend: Node.js, Express.js, Socket.IO, JWT

Database: MongoDB Atlas, Mongoose

Media Storage: Cloudinary

Deployment: Render

 Learning Outcomes

This project strengthened my understanding of REST API development, JWT authentication, WebSocket communication using Socket.IO, state management with Redux Toolkit, MongoDB schema design, cloud media storage, production deployment, and debugging real-world full-stack applications.
